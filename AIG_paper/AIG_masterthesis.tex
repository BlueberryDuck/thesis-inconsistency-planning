\englishabstract{
Classical planning systems excel at finding solutions when they exist, but provide minimal diagnostic information when encountering unsolvable problems, forcing users into costly trial-and-error debugging processes. This thesis investigates the adaptation of inconsistency measures from propositional logic to provide quantitative diagnostics for unsolvable planning instances. Motivated by the finding that naive propositional encodings of planning problems yield uninformative measure values, this work develops planning-native diagnostic profiles that operate directly on planning structures: goals, operators, and precondition-effect relationships. Three complementary measures are proposed: a goal conflict profile for detecting specification-level mutex relationships, a goal sequencing conflict profile for identifying achievability-level conflicts caused by action effects, and an unreachability profile for characterizing dependency structure failures. The measures are implemented using Answer Set Programming, building on findings that ASP-based approaches show promise for inconsistency computation. Experimental evaluation demonstrates that the adapted measures can distinguish between different structural categories of unsolvable planning problems and provide actionable diagnostic information for problem repair, addressing a methodological gap in automated planning tools.
}
\germanabstract{
Klassische Planungssysteme sind hervorragend darin, Lösungen zu finden, wenn sie existieren, aber bieten minimale diagnostische Informationen bei unlösbaren Problemen, was Nutzende zu kostspieligem trial-and-error Debugging zwingt. Diese Arbeit untersucht die Anpassung von Inkonsistenzmaßen aus der Aussagenlogik, um quantitative Diagnostiken für unlösbare Planungsinstanzen bereitzustellen. Motiviert durch die Erkenntnis, dass naive aussagenlogische Kodierungen von Planungsproblemen uninformative Messwerte liefern, entwickelt diese Arbeit planungsnative diagnostische Profile, die direkt auf Planungsstrukturen operieren: Ziele, Operatoren und Vorbedingung-Effekt-Beziehungen. Drei komplementäre Maße werden vorgeschlagen: ein Zielkonflikt-Profil zur Erkennung von Mutexbeziehungen auf Spezifikationsebene, ein Zielsequenzkonflikt-Profil zur Identifikation von Erreichbarkeitskonflikten durch Aktionseffekte, und ein Unerreichbarkeits-Profil zur Charakterisierung von Abhängigkeitsstrukturfehlern. Die Maße werden mittels Answer Set Programming implementiert, aufbauend auf Erkenntnissen, dass Ansätze basierend auf ASP vielversprechend für Inkonsistenzberechnungen sind. Die experimentelle Evaluation zeigt, dass die adaptierten Maße zwischen verschiedenen strukturellen Kategorien unlösbarer Planungsprobleme unterscheiden können und handlungsrelevante diagnostische Informationen für Problemreparatur liefern, wodurch eine methodische Lücke in automatisierten Planungstools adressiert wird.
}
