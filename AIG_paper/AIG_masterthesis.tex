\englishabstract{
Classical planning systems excel at finding solutions when they exist, but provide minimal diagnostic information when encountering unsolvable problems, forcing users into costly trial-and-error debugging processes. This thesis investigates the adaptation of inconsistency measures from propositional logic to provide quantitative diagnostics for unsolvable planning instances. The research explores how selected inconsistency measures can be systematically adapted to classical planning domains, addressing the fundamental challenges posed by temporal action sequences and state transitions that distinguish planning from static knowledge bases. The adaptation approach employs Answer Set Programming (ASP) implementations, building on recent findings that ASP-based approaches show promise for inconsistency computation. This work aims to develop a systematic framework for quantifying inconsistencies in planning instances and explore whether adapted measures can distinguish between different types of unsolvable problems. The goal is to investigate whether such diagnostic capabilities can help users identify conflicting constraint sets and provide insights for problem repair, potentially addressing a methodological gap in automated planning tools.
}
\germanabstract{
Klassische Planungssysteme sind hervorragend darin, Lösungen zu finden, wenn sie existieren, aber bieten minimale diagnostische Informationen bei unlösbaren Problemen, was Nutzende zu kostspieligem trial-and-error Debugging zwingt. Diese Arbeit untersucht die Anpassung von Inkonsistenzmaßen aus der Aussagenlogik, um quantitative Diagnostiken für unlösbare Planungsinstanzen bereitzustellen. Die Forschung erforscht, wie ausgewählte Inkonsistenzmaße systematisch auf klassische Planungsdomänen angepasst werden können, wobei die fundamentalen Herausforderungen durch zeitliche Aktionssequenzen und Zustandsübergänge adressiert werden, die Planung von statischen Wissensbasen unterscheiden. Der Anpassungsansatz verwendet Answer Set Programming (ASP) und baut auf neueren Erkenntnissen auf, dass ASP-basierte Ansätze vielversprechend für Inkonsistenzberechnungen sind. Diese Arbeit zielt darauf ab, ein systematisches Framework zur Quantifizierung von Inkonsistenzen in Planungsinstanzen zu entwickeln und zu erforschen, ob angepasste Maße zwischen verschiedenen Typen unlösbarer Probleme unterscheiden können. Das Ziel ist zu untersuchen, ob solche diagnostischen Fähigkeiten Nutzenden helfen können, konflikthafte Constraint-Mengen zu identifizieren und Einsichten für Problemreparatur bereitzustellen, wodurch möglicherweise eine methodische Lücke in automatisierten Planungstools adressiert wird.
}
