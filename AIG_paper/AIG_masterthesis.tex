\englishabstract{
Classical planners report only that a problem is unsolvable, leaving users without information about why no plan exists. This thesis adapts inconsistency measures from propositional logic to classical planning, providing diagnostics that characterize unsolvability structure. Encoding planning problems as propositional knowledge bases and applying the contension measure yields uninformative results, motivating measures that operate directly on goals, operators, and reachable states. Three planning-native measures are proposed. The Unreachability Profile counts goal propositions absent from all reachable states. The Goal Mutex Profile detects operator-induced mutual exclusions where goals are individually achievable but never satisfiable in a common reachable state. The Goal Sequencing Conflict Profile identifies irreversible exclusions where achieving one goal permanently blocks another. Each measure is horizon-parameterized, analyzed against planning-adapted rationality postulates, and shown to be PSPACE-complete. The measures are implemented in Answer Set Programming using \textit{plasp} and Clingo brave reasoning. An experimental evaluation on the IPC~2016 Unsolvability Competition and the Eriksson benchmark collection shows that the profiles distinguish structural categories of unsolvable instances and reveal a soundness-coverage tradeoff inherent to horizon-bounded analysis.
}
\germanabstract{
Klassische Planungssysteme melden, dass ein Problem unlösbar ist und liefern keine Informationen darüber, warum kein Plan existiert. Diese Arbeit überträgt Inkonsistenzmaße aus der Aussagenlogik auf die klassische Planung und stellt Diagnostiken bereit, die die Struktur der Unlösbarkeit charakterisieren. Eine Kodierung von Planungsproblemen als aussagenlogische Wissensbasis mit Anwendung des Contension-Maßes liefert uninformative Ergebnisse und motiviert Maße, die direkt auf Zielen, Operatoren und erreichbaren Zuständen operieren. Drei planungsnative Maße werden vorgeschlagen. Das Unerreichbarkeitsprofil zählt Zielpropositionen, die in keinem erreichbaren Zustand vorkommen. Das Zielmutex-Profil erkennt operatorinduzierte gegenseitige Ausschlüsse, bei denen Ziele einzeln erreichbar sind, jedoch nie gleichzeitig erfüllbar sind. Das Zielsequenzierungskonflikt-Profil identifiziert irreversible Ausschlüsse, bei denen das Erreichen eines Ziels ein anderes dauerhaft blockiert. Jedes Maß ist durch einen Horizont parametrisiert, wird gegen planungsadaptierte Rationalitätspostulate analysiert und ist PSPACE-vollständig. Die Maße werden mittels Answer Set Programming unter Verwendung von \textit{plasp} und Clingo-Brave-Reasoning implementiert. Eine experimentelle Auswertung auf Benchmarks der IPC~2016 Unsolvability Competition und der Eriksson-Sammlung zeigt, dass die Profile strukturell unterschiedliche Kategorien unlösbarer Instanzen unterscheiden, und offenbart ein Spannungsverhältnis zwischen Korrektheit und Abdeckung, das der horizontbeschränkten Analyse inhärent ist.
}
