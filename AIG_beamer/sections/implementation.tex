\begin{frame}[fragile]{Witness Extraction and Profile Construction}
    The reachability encoding derives \texttt{holds(P, T)}, true when proposition $P$ holds at step $T$. One witness rule per measure reads it.

    \begin{block}{Witness rules}
        \begin{lstlisting}[style=asp]
% True reachability: P in some reachable state
true_reachable(P) :- holds(P, T), prop(P).
% Coexistence: G1 and G2 hold in one state
coexist_witness(G1, G2) :- goal(G1), goal(G2), G1 < G2,
    holds(G1, T), holds(G2, T), time(T).
% Sequencing: G2 reachable after G1
g2_after_g1_witness(G1, G2) :- goal(G1), goal(G2), G1 != G2,
    holds(G1, T1), holds(G2, T2), T2 >= T1.
        \end{lstlisting}
    \end{block}

    {\small Brave unions these across traces, so a \emph{missing} witness is a universal proof that \emph{no} trace satisfies the condition. Eleven hand-crafted scenarios reproduce their expected profiles exactly.}
